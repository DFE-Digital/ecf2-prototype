The current page width for examples within the design framework is set to 1020px.

This is the default maximum width as specified by the GOV.UK Design System. However, it can be adjusted where there's a need to increase the width of the page, if the content requires it.

This may get adjusted in the future for that purpose.

To adjust the page width for all examples and pages within the design framework, you can update the govuk-width-container class in the layouts/plain.html file.

For example, to set the page width to 1200px, you can update the govuk-width-container class to:

.govuk-width-container 
{
    max-width: 1200px;
}
