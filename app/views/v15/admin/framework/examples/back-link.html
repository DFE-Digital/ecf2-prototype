This example shows how to use the back link component to help users navigate back to the previous page. We'd use this when users need a simple way to return to where they came from in a linear journey.

Back links are most useful in step-by-step processes, forms, or when users are viewing detailed information and need to return to a list or overview page. They should be placed consistently at the top of the page, before the main heading.

If the user needs to return to a previous page and see the information they were looking at in the same state, we'd use a back link instead of breadcrumbs.
