Round 1 user research

This version contains a happy path for adding an ECT for a school user.

This includes collecting TRN & DoB, to play back the name from the teacher record. It also includes collecting 'default' programme information if a school hasn't added previously, or playing back 'default' programme information if they have.

Adding a mentor journeys, non-standard inductions, homepages / sign in flows, start dates and adding using checklists / save and return are all out of scope in this prototype version.
